## Stacks

**Front-end:** React Native, Styled Components

## Run

Clone the project

```bash
  git clone https://github.com/LuisLenzi/instagram-clone.git
```

Change the folder

```bash
  cd instagram-clone
```

Install the dependencies

```bash
  npm install
```

Build the project

```bash
  npm run build
```

Run the server

```bash
  npm run start
```
